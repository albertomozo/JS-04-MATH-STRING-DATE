Substring

El método indexOf() devuelve la posición de la primera aparición de un valor especificado en una cadena.
Este método devuelve -1 si el valor de búsqueda núnca ocurre.
Es sensible a mayúsculas.
El índice del primer carácter es el 0, el segundo es 1 y así sucesivamente.
*** sirve para validar formularios, o emails (si tiene o no tiene @, para teléfonos...) = textos
